# 项目介绍短文本

## 项目名称

MXMoE-Adapt：面向沐曦曦云C500的路由负载感知MoE融合算子协同优化与自适应派发

## 项目简介（约200字）

本项目面向沐曦曦云C500与MXMACA软件栈，研发路由负载感知的MoE融合算子协同优化与自适应派发系统。项目针对通用MoE算子固定参数难以适应Prefill/Decode、专家负载偏斜和软件版本变化的问题，联合优化路由对齐块、专家GEMM分块与运行时配置选择，建立C500环境指纹、离线自动调优器、已验证配置数据库、置信回退和性能漂移检测机制。项目将完成FP16/BF16单卡C500实测、真实稀疏MoE模型验证及FlagGems MetaX后端开源贡献。

## 项目创新性（约150字）

项目从传统静态Shape调优升级为动态路由分布感知调优，引入路由熵、负载偏斜、空专家比例和对齐填充率；将路由对齐块与GEMM Tile、warps和stages联合搜索；配置只在匹配C500环境指纹且经过真机验证时派发，超出覆盖范围自动回退；当MXMACA或编译栈变化导致性能漂移时自动触发重新调优。

## 预期成果（约150字）

形成一个Apache-2.0开源仓库，交付C500环境采集工具、MoE路由特征与Shape基准、联合自动调优器、C500专用配置数据库、运行时派发与漂移检测模块、FP16/BF16正确性和性能报告、真实MoE模型验证示例，并向FlagGems提交至少一个MetaX后端Issue或PR，项目同步镜像至GitLink。

## 项目关键词

沐曦；曦云C500；MXMACA；FlagGems；MoE；Fused MoE；Triton；算子优化；国产算力；大模型推理
